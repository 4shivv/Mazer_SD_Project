Mazer is a secure, fully offline Al assistant built using React, Node.js, MongoDB, ChromaDB, Docker, and a locally hosted LLaMA model via Ollama.

The system implements Retrieval-Augmented
Generation (RAG), JWT-based authentication, role-based access control, secure document ingestion, real-time token streaming, GPU-aware session management, and structured system monitoring.

Designed for air-gapped environments requiring secure, high-availability Al systems.
